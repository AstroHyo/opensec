# Core Beliefs

## 1. Deterministic First

The system should know what it is looking at before asking a model to say something smart about it.

## 2. Evidence Before Eloquence

A plain but correct digest beats a polished but weakly grounded digest.

## 3. OpenAI Signal Deserves Special Handling

Official OpenAI updates are a first-class priority for this user and should rarely be buried.

## 4. Practical Signal Beats Hype

We value workflow changes, methodology, developer ergonomics, and meaningful OSS over generic attention bait.

## 5. State Matters

A useful briefing system remembers what it has shown, why it chose it, and how to answer follow-up questions later.

## 6. LLMs Should Improve Judgment Expression, Not Replace System Memory

Models should help compress, explain, and synthesize, while the system retains retrieval, ranking controls, and historical state.

## 7. Graceful Degradation Is A Feature

If a source is down or a model fails, the system should still produce the best possible digest from the remaining evidence.
