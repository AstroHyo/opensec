# Completed Plans

Move finished execution plans here once the work has shipped and the behavior is documented elsewhere.

Archive rules:

- if a plan has no real remaining execution step, it should not stay in `active/`
- active plans should keep a short status header with:
  - `Live status`
  - `Next step`
  - `Owner`
- if new work starts after a plan is completed, open a new active plan instead of reactivating the archive by default
