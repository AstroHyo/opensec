# memory/

Use this folder for raw, short-lived, and date-scoped context.

## File shape

- `YYYY-MM-DD.md`: daily note for raw memory capture

## Daily note sections

- `## Snapshot`
  - high-level summary of the day
- `## Discord Conversation Notes`
  - short bullets from meaningful Discord exchanges
- `## Candidate Signals`
  - signals that may become durable memory
- `## Promotion Candidates`
  - distilled items that are ready for owner review
- `## Promoted to MEMORY`
  - audit trail of what actually moved into `MEMORY.md`

## Rules

- Do not dump whole transcripts here.
- Do not put secrets here.
- Prefer short bullets over narrative logs.
- Shared guild-channel context can be noted here.
- Private DM details should only be noted when the owner explicitly wants them retained.

`MEMORY.md` should remain smaller and more durable than this folder.
